DropBox is a cloud data backup and sync service.


7/10


Hosted in by Dropbox, Inc.

Needs fixing

  • Does not support federated logins that allow you to login with centralised credentials such as Google or Facebook
  • Uses fixed questions and user-provided answers for password resets
  • Unknown whether or how encryption is used for storing data
  • Indefinate storage of user data
  • No public disclosure policy that states how it handles vulnerability reports from security researchers
  • No bug bounty programm that encourages responsible disclosure of vulnerabilites and pays researchers for their efforts
  • May user personal data for marketing purposes
  • May pass on personal data to third parties
  • Has had a recently securit incident in while sensitive user data mwas made public by unauthorized third parties

A job well done

  • Allows two-factor (e.g. SMS or mobile app)
  • Stores account passwords securely
  • Unrestricted password character set and/or length
  • Small amount of personal or sensitive data is stored
  • All traffic to site is encrypted
  • SSL/TLS is rated as A by Qualsys SSLabs
